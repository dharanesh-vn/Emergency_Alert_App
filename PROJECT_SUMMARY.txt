HOW TO BUILD & RUN
------------------

OPTION 1: Android Studio (Recommended)
1. Open Android Studio
2. File → Open → Select "EmergencyAlertApp" folder
3. Wait for Gradle sync (first time: 5-10 minutes)
4. Click Run button (▶️)
5. Select emulator or connected device

OPTION 2: Command Line
1. Navigate to EmergencyAlertApp folder
2. Run: ./gradlew assembleDebug
3. APK location: app/build/outputs/apk/debug/app-debug.apk
4. Install: adb install app/build/outputs/apk/debug/app-debug.apk

OPTION 3: Build APK in Android Studio
1. Build → Build Bundle(s) / APK(s) → Build APK(s)
2. Find APK in: app/build/outputs/apk/debug/
3. Transfer to device and install
